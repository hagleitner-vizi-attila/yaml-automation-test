Hsm.BaseTypes

The Hsm.BaseTypes package contains a collection of types. These types find use in various different projects.

Eui64Definition

This type represents a standardized address format (EUI-64).

ArticleNumberDefinitions

SalesProductArticleNumber

This type represents a sales product article number.

VersionDefinitions

IntegerVersion

This type represents an integer version number.

SemanticVersion

This type represents a semantic version number.

Parts of a semantic version number
  • Major version number
  • Minor version number
  • Patch version number
  • Pre-release identifier
  • Build identifier
Examples
  • 1.0.1
  • 1.0.1-alpha
  • 1.0.1-alpha+001

SerialNumberDefinitions

SerialNumberV1

This type represents a version 1 Hagleitner serial number.

Format

YYMMDDEE-SSS+

  • YY - year
  • MM - month
  • DD - day
  • EE - employee number (FixedLengthNumber)
  • S+ - at least 1 digit of continuous sequential number (FixedLengthNumber)

SerialNumberV2

This type represents a version 2 Hagleitner serial number.

Format

TTTT-YYMMDD-SSSSS

  • TTTT - type, production line, machine id, or test device (FixedLengthNumber)
  • YY - year
  • MM - month
  • DD - day
  • SSSSS - continuous sequential number (FixedLengthNumber)

SerialNumberV3

This type represents a version 3 Hagleitner serial number.

Format

YYMMDDLLLSSSS

  • YY - year
  • MM - month
  • DD - day
  • LLL - production line, machine id, or test device (Base36 encoding)
  • SSSS - continuous sequential number (Base36 encoding)

NumberDefinitions

FixedLengthNumber

This type represents a fixed length number. For example 42 with length of 3 would result in 042.

Base36Cipher

This type represents a number with a base defined by 0-9 and letters a-z (based on ASCII characters).

Intervals

DoubleInterval

This type represents a double interval. The interval can be left- and / or right-open.

DateTimeOffsetInterval

This type represents a date interval (with DateTimeOffset). The interval can be left- and / or right-open.

AbsoluteDateInterval

This type represents an absolut date interval with a start date as DateTimeOffset and an end date as DateTimeOffset.

RelativeDateInterval

This type represents a relative date interval. Start and end date are calculated by offsetting the current date with provided offset classes.

Validatable

Implements validation using the notification pattern.

ValidationResult

This type represents the result of a validation run produced by objects implementing the Validatable interface.

ValidationMessage

This type represents a message which contains validation information.

Severity

Defines the level of severity that a ValidationMessage could have.

ValidationException

This exception can be thrown if a ValidationResult contains at least one error level ValidationMessage.

ValidationResultFormatter

Represents a class to format a ValidationResult.

ValidationResultTranslator

Represents a translator, which translates ValidationMessage of a ValidationResult.
