TryParse(String, out SemanticVersion)

Converts the string representation of an SemanticVersion to its SemanticVersion equivalent. A return value indicates whether the operation succeeded.

Declaration
public static bool TryParse(string semanticVersionString, out SemanticVersion semanticVersion)
Parameters
Type Name Description
System.String semanticVersionString

A string representing a semantic version which shall be converted.
SemanticVersion semanticVersion

When this method returns, contains the SemanticVersion equivalent of the semantic version contained in semanticVersionString, if the conversion succeeded, or null if the conversion failed. The conversion fails if the semanticVersionString parameter is null or Empty, or is not of the correct format. This parameter is passed uninitialized; any value originally supplied in semanticVersion will be overwritten.
Returns
Type Description
System.Boolean

True if semanticVersionString was converted successfully; otherwise, false.
