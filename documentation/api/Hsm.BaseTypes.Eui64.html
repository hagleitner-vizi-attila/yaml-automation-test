TryParse(String, out Eui64)

Converts the string representation of an Eui64 to its Eui64 equivalent. A return value indicates whether the operation succeeded.

Declaration
public static bool TryParse(string address, out Eui64 eui64)
Parameters
Type Name Description
System.String address

A string containing a address to convert.
Eui64 eui64

When this method returns, contains the Eui64 equivalent of the address contained in address, if the conversion succeeded, or an Eui64 with all zeros as its address if the conversion failed. The conversion fails if the address parameter is null or Empty, or is not of the correct format. This parameter is passed uninitialized; any value originally supplied in eui64 will be overwritten.
Returns
Type Description
System.Boolean

True if address was converted successfully; otherwise, false.
